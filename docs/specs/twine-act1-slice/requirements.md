# Requirements

Build a Twine/SugarCube demo that routes the full released Act 1 public corpus.

## User Experience

- Player starts at a simple Act 1 entry passage with author byline.
- Player can follow the main Act 1 spine from prologue through chapter 20.
- Player can branch into every lettered optional Act 1 scene at the relevant
  chapter.
- Player can use a route map to jump directly to any public Act 1 scene.

## Constraints

- Single generated HTML file.
- Full public Act 1 only: main chapters plus lettered optional scenes.
- No later-act content.
- No dependency on Ren'Py or Godot code.
- Deterministic release gate must pass locally.
